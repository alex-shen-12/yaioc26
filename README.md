# **赛题介绍**

本赛题要求构建颜料样品老化预测模型，利用颜料短期的紫外老化表现，预测长期老化下的色差。

# **数据说明：**

1.  训练集：

训练集包含未发表的实验测量的181条颜料老化数据，颜料的老化程度通过色度进行测量，色度一般通过Lab颜色空间表示。Lab色空间是一种颜色表示方法，由三个通道组成：

+   L\* 亮度通道：亮度（0黑 → 100白）
+   a\* 通道：红（正）→ 绿（负）
+   b\* 通道：黄（正）→ 蓝（负）
+   **ΔE**：表示在Lab空间中两个颜色之间的欧氏距离：  
    **ΔE\=(ΔL∗)2+(Δa∗)2+(Δb∗)2​**

在老化实验中，研究者首先按照一定规范制备样品，并测量其原始的Lab值（`L0`, `a0`, `b0`）。在一定老化条件下老化数天后，测量当天的Lab值（`L`, `a`, `b`），并和`L0`, `a0`, `b0`求色差。通过测量多天的色差值即可判断老化速度。一般来说，ΔE<1 时人眼难以看出差异，ΔE\>2 时通常能明显看出色差。

每条数据包含的内容如下：

| 字段名 | 类型 | 含义 |
| --- | --- | --- |
| `sample` | string | 样品名称，标记了颜料名称和样品编号，不同样品间可能存在制备上的细微差异 |
| `aging_condition` | string | 老化条件，UV表示紫外照射老化，humid-\_heat表示湿热条件老化 |
| `aging_time_day` | int | 老化时间（单位：天） |
| `L0` | float | 老化前亮度通道（Lab 中的 L\*） |
| `a0` | float | 老化前红绿通道（Lab 中的 a\*） |
| `b0` | float | 老化前黄蓝通道（Lab 中的 b\*） |
| `L` | float | 老化后亮度通道（L\*） |
| `a` | float | 老化后红绿通道（a\*） |
| `b` | float | 老化后黄蓝通道（b\*） |
| `dietaE` | float | 老化前后综合色差（ΔE） |

其中同一样品的`L0`、`a0`、`b0`值应完全相同

2.  测试集：

测试集总共包含37条数据，部分为训练集中样品在更长老化时间下的色度，部分数据为全新样品的色度老化曲线

选手应基于训练集训练模型后，根据测试集给定的样品名、老化条件、`L0`、`a0`、`b0`值等，预测老化时间下该样品的色差值。

| 字段名 | 类型 | 含义 |
| --- | --- | --- |
| `sample` | string | 样品名称，标记了颜料名称和样品编号，不同样品间可能存在制备上的细微差异 |
| `aging_condition` | string | 老化条件，UV表示紫外照射老化，humid-\_heat表示湿热条件老化 |
| `aging_time_day` | int | 老化时间（单位：天） |
| `L0` | float | 老化前亮度通道（Lab 中的 L\*） |
| `a0` | float | 老化前红绿通道（Lab 中的 a\*） |
| `b0` | float | 老化前黄蓝通道（Lab 中的 b\*） |

# **评测指标**

色差预测值和实验测量值计算RMSD：

$$RMSD = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (\Delta E_i^{pred} - \Delta E_i^{exp})^2}$$

后线性转化成百分制
